  2. Jiacheng Xue, Xiang Chen*, Zhanqi Cui, Yong Liu.
    Learning Never Stops: Improving Software Vulnerability Type Identification via Incremental Learning and Long-tailed Learning.
    (Under Review, CCF-B, Q2 Journal)
    Abstract: This study addresses challenges in SVTI with incremental and long-tailed learning using the {\tool} framework. Techniques include hybrid replay, EWC-based regularization, focal and smoothed loss. Prompt tuning with CodeT5 and hybrid inputs is applied. Evaluation on a 6,269-vulnerability dataset shows state-of-the-art performance over baselines like VulExplainer and LIVABLE.
